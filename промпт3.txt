Сгенерируй секцию “Мой индекс навыков”.

Требования:
1. HTML:
   – заголовок  “Мой индекс навыков”,
   – 5 input type="range" (HTML, CSS, JS, React/Vue, LLM/Нейросети),
   – под каждым — вывод значения (%) через .
2. JavaScript:
   – при изменении любого слайдера обновляется среднее значение,
   – показывается итоговый уровень (в %),
   – рядом текстовое описание (на основе диапазона):
       0–25% → “Начинающий, верный старт”,
       26–50% → “Прогрессируешь, держи темп”,
       51–75% → “Уверенно растёшь, почти junior”,
       76–100% → “Сильная база — готов к портфолио”.
   – все значения сохраняются в localStorage.
3. Визуал:
   – индикатор-гейдж (progress bar) с плавным заполнением,
   – минималистичный дизайн, чистый чёрно-белый с акцентом.
4. Код на чистом HTML + CSS + JS, без фреймворков.
