Сгенерируй hero-секцию для личного сайта начинающего разработчика.

Требования:
1. Используй чистый HTML + CSS (без фреймворков).
2. Внутри блока должны быть:
   – имя пользователя (в виде ),
   – короткий bio (1–2 предложения в ),
   – подзаголовок “Мой путь в IT + AI”.
3. Добавь одну кнопку CTA (“Мои навыки”).
4. Стили: минимализм, центрирование контента, адаптивность.
   – фон — тёмный (#0b0b0b) с лёгким градиентом,
   – текст — светлый,
   – акцент-цвет — через переменную --accent.
5. Добавь лёгкую анимацию появления текста при загрузке (через opacity + translateY).
6. Сделай код читаемым, понятным новичку.
7. Структура проекта строго:
    – index.html
    – styles.css
    – script.js
